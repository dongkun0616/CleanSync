# CleanSync
남서울대학교 리버스 동아리 - 실시간 실내 공기질 및 혼잡도 모니터링 시스템 [Clean-Sync]

1. 프로젝트 개요
프로젝트 배경 : [코로나19 이후 실내 환기의 중요성이 커졌으며, 시험 기간 중 도서관이나 강의실의 혼잡도를 미리 파악하여 쾌적한 학습 환경을 선택하고자 하는 니즈가 발생함.]
프로젝트 목적 :	[IoT 센서를 통해 강의실의 미세먼지, 이산화탄소 농도, 소음 정보를 수집하고, 이를 시각화하여 학생들에게 실시간 학습 환경 정보를 제공하는 것.]
프로젝트 기간 : 2026년 3월 ~ 2026년 6월 (약 3개월)

2. 기술 스택 
Frontend	[React.js]
Backend	[Node.js (Express)]
Database	[MySQL]
배포/인프라	[AWS(서버), MQTT Broker(데이터 통신)]
협업 도구	[GitHub, Notion, Discord]
